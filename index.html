Horror Game Framework (UE5)

A first-person horror gameplay framework built in Unreal Engine 5, focused on modular systems and core gameplay architecture.

Screenshots

Key Features

  • Full gameplay framework setup (Game Mode, Player Controller, HUD, Player State)
  • First-person character controller (movement, sprinting, crouching, jumping)
  • Interaction system (doors, objects, physics-based interactions)
  • Inventory system with item examination and usage
  • Survival mechanics (flashlight, battery, health system)
  • Atmospheric horror systems (jumpscares, objectives, notes, hiding)
  • AI enemy system (patrol, chase, investigate, attack states)
  • Advanced mechanics (security cameras, cutscenes, leaning system)

Built using Unreal Engine 5 Blueprints with a focus on reusable gameplay systems.

MicroCars (UE5)

An arcade racing prototype featuring miniature vehicles navigating stylized themed levels. The project focuses on vehicle gameplay, UI integration, and overall player experience.

Screenshots

Key Features

  • Vehicle movement system with acceleration, braking, and drifting
  • Multiple themed levels with unique environments
  • Real-time driving HUD displaying speed and gear information
  • Main menu and game state management
  • Arcade-focused vehicle handling and gameplay tuning

Built in Unreal Engine 5 with a focus on responsive controls, level variety, and accessible arcade gameplay.
